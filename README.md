# Landform-Specific-Land-Surface-Temperature-Analysis-in-the-Teesta-River-Using-Landsat-and-MODIS
Reproducible Google Earth Engine workflow for landform specific land surface temperature analysis in the Teesta River Bangladesh using Landsat 8 9 Collection 2 Level 2 and MODIS MOD11A2 data from 2019 to 2024 with pixel diagnostics seasonal aggregation and sensor comparison.
